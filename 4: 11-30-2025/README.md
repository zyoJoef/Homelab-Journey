<p>As I was casually browsing on my Facebook Marketplace, I've stumbled across a listing of an HP 2530 8g PoE+ J9774A managed switch which I got for a good deal to replace the old unmanaged switch. Being a fanless model, I wasn't expecting it to run hot even without load (more so when I added load/s into it). The great thing I found about it is that it has the typical two light indicator on the port instead of just the small and single one that is present on the unmanaged switch that I have. Though to be honest, I don't even make use all the features it has (such as the PoE, VLAN, etc.) aside from the web interface.</p>
